RandomUserList presentation logic.

  • presentFetchRandomUsers(_:currentPage:error:)

    Presents to the view controller the random users with the fetched random users.

    If there are no more users to retrieve the provided users sequence should be the same, the current page will be the previous page from the one provided (not less than 0), and a noMoreUsersToLoad error will be returned.

    The sequence of users to be shown.
    currentPage

    error

    A problem that has occurred during the random users fetch.
  • presentFilterRandomUsers(_:appliedFilter:)

    Presents to the view controller the provided random users with the defined filter applied. If the filter has no matches to the random users provided, an empty sequence will be returned.
